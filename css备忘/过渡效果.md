### transition

语法格式

```
transition：要过渡的属性 话费时间 运动曲线 合适开始
如果有多组属性变化，用逗号隔开
```



| **属性**         | **是否脱标占有位置** | **是否可以使用边偏移** |
| ---------------- | -------------------- | ---------------------- |
| 静态static       | 不脱标，正常模式     | 正常模式               |
| 相对定位ralative | 不脱标，占有位置     | 相对自身位置移动       |
| 绝对定位absolute | 完全脱标，不占有位置 | 相对于定位父级移动位置 |
| 固定定位fixed    | 完全脱标，不占有位置 | 相对于浏览器移动位置   |




```css
div{
    width: 100px;
    height: 200px;
    background: #ccc;
    transition: width 0.5s
}
div:hover{
    width: 600px;
}
```
