### 隐藏元素

```css
div:{
    display:none; /*不占有位置*/
}
div:{
    visibility:hidden; /*隐藏之后还占有位置*/
}
```

### 鼠标经过div显示二维码的案例

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta http-equiv="X-UA-Compatible" content="ie=edge">
  <title>Document</title>
  <style>
    div{
      width: 100px;
      height: 100px;
      background-color: #ccc;
      text-align: center;
      margin: 100px auto;
      position: relative;
    }
    div img{
      position: absolute;
      left: 100px;
      top: 0;
      display: none;
    }
    div:hover img{ /* 鼠标经过div的时候，div里面img元素的样式 */
      display: block;
    }
  </style>
</head>
<body>
  <div>
    扫二维码
    <img src="./qrcode.jpg" alt="">
  </div>
</body>
</html>
```
